# ALX Travel App 0x01

## API Endpoints

The project provides RESTful API endpoints for Listings and Bookings using Django REST Framework.

### Endpoints

- /api/listings/
- /api/listings/<id>/
- /api/bookings/
- /api/bookings/<id>/

Each endpoint supports:
- GET
- POST
- PUT
- DELETE
